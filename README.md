# typescript-eslint-example

Typescript-eslint lint - Usage example

[![LICENSE](https://img.shields.io/github/license/henriquecarv/typescript-eslint-example.svg)](./LICENSE)

## System Requirements

- **[NodeJS](https://nodejs.org/en/)** (version >= 10).

## Installing

- npm install

## Compiling

- npm start

## Lint Testing

- npm run lint
